An Inspector Calls - Audiobook

On this page, you will find the audiobook for An Inspector Calls. There are six parts. Each part is exactly the same as the reading parts.

Audiobook

Navigate to the part you want to hear. You are on a recording if you hear Audio Start. If you hear this, go down one time and you will hear Wiedergeben Schalter. Activate the button by pressing the spacebar. Now you will hear the recording. You can stop it and start it again with the spacebar. If the recording is finished, you can go down until you hear Audio Ende. If you hear this, you are no longer on the audio control menu.
