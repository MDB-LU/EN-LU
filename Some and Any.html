Some and Any

We use some and any if we don't know how many items of a thing there are, or if it is not relevant to say how many items there are.

We also use some and any with uncountable nouns (for example: water, money, sand).

Examples:

I don't want any presents for my birthday.

There are some books on the shelf.

Do you have any apples?


Some

Some is generally used in positive sentences.

Examples:

There are some eggs in the fridge.

Some fans didn't support their team.

We still have some milk left.


Any

Any is generally used in negative sentences and in questions.

Examples:

He doesn't have any chocolate.

Do you have any pictures at home?

There hasn't been any real improvement.


IMPORTANT

There is an exception to the rule. It is also possible to use some in questions if you want to offer something to someone or if you want to request something from someone.

Examples:

Would you like some water?

Could I have some water?
