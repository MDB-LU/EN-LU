There is - there are

If we want to describe what we can see, we can use the words there is and there are.

Examples:

There is a mouse in the basement.

There are three televisions in our house.

There are some books on the floor.


We use there is if there is only one item. (singular)

We use there are if there are multiple items. (plural)


Negatives

We can also make negative sentences if we don't see an item. In that case, we have to add the word not.

Examples:

There is not a computer. OR There isn't a computer.

There are not any photographs. OR There aren't any photographs.


Questions

We can also ask questions. In that case, we just have to change the order of there is to is there, and from there are to are there.

Examples:

Is there a swimming pool?

Are there any magazines?
