Present Perfect Continuous

We use the present perfect continuous to talk about a finished activity in the recent past. Using the present perfect continuous focuses on the activity.

Even though the activity is finished, we can see the result in the present.

We make the Present Perfect Continuous with have been and the ing form of the main verb.

Examples:

I have been cleaning the car. (Present Result: The car is now wet and clean.)

It has been snowing. (Present Result: The ground is now covered in snow.)


Negative

If we want to make a negative sentences in the Present Perfect Continuous, we need to add the word not. This word comes between have and been.

Examples:

I have not been feeling well.

He has not been paying attention.


Questions

If we want to ask a question in the Present Perfect Continuous, we have to put the auxiliary verb have in front of the subject.

Examples:

Have you been working a lot lately?

What have you been thinking about?


IMPORTANT

We can use the Present Perfect Continuous for one continuing event that began in the past and is still going on.

Example:

He has been living in London since 1995. (He is still living in London).


We can also use the Present Perfect Continuous to talk about repeated activities which started in the past and are still going on.

Example:

They have been going to Spain on holiday every year since 1987.
